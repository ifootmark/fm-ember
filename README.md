# fm-ember

## 安装依赖

安装  node 和 npm

Node.js 版本 > 0.12 , npm 版本 > 2.7 , 查看版本。

```javascript
node -v
npm -v
```

参考：
[Node.js](https://nodejs.org/)，
[npm](https://www.npmjs.com/)

安装 Bower Web包管理器

`npm install -g bower`

安装 Git

[Git](http://git-scm.com/download/win)


## 运行项目

`git clone https://github.com/ifootmark/fm-ember.git`

`npm install`

`npm run lite-cli`

`npm run lite`
